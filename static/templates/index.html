About the System

This Multi-Disease Prediction System is designed to help healthcare professionals predict various diseases based on patient data. The system uses machine learning algorithms to analyze patterns in the data and make predictions.

Supported Diseases

How It Works

  1. Data Input: Enter patient data for the selected disease.
  2. Preprocessing: The system handles missing values and outliers.
  3. Prediction: Machine learning models analyze the data and make predictions.
  4. Results: View the prediction results with probability scores.

Technologies Used

  • Backend: Python, Flask, Scikit-learn
  • Frontend: HTML, CSS, JavaScript
  • Deployment: Docker
